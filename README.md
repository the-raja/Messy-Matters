## 🎯 Objective

To build a **Smart Mess Feedback & Menu Portal** that allows students to share feedback easily and helps wardens analyze mess performance efficiently.

---

## 💡 Solution Overview

**Messy Matters** is a web-based application where:
- Students can view mess menus and submit feedback
- Wardens can monitor ratings and complaints
- Mess menu can be updated digitally in real time

---

## ✨ Features

### 👩‍🎓 Student Side
- View **daily & weekly mess menu**
- Rate meals:
  - Breakfast
  - Lunch
  - Snacks
  - Dinner
- Submit feedback:
  - Anonymous
  - With name
- Simple and user-friendly interface

---

### 🧑‍💼 Warden Dashboard
- Secure login (**Password: `warden123`**)
- View:
  - Average ratings
  - Frequent complaints
  - Meal-wise comparison
- Update mess menu
- Separate **public** and **admin** views

---

## ✅ Benefits

- Organized and structured feedback system
- Encourages honest feedback through anonymity
- Helps wardens take **data-driven decisions**
- Improves food quality and student satisfaction
- Replaces unstructured chat-based complaints

---

## 🛠️ Technology Stack

- **Frontend:** Next.js, React
- **Backend:** Next.js API Routes
- **Database:** MongoDB Atlas (with local JSON fallback)
- **Charts:** Chart.js
- **Deployment:** Vercel

---

## 🏁 Conclusion

**Messy Matters** converts unorganized hostel mess complaints into **clear, actionable insights**, improving transparency, accountability, and overall dining experience.
