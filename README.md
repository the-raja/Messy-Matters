# Messy Matters – Smart Hostel Mess Feedback & Menu Viewer

---

## Problem Statement

Hostel students often complain about mess food quality, monotonous menus, or sudden menu changes.  
Currently, this feedback is scattered across informal channels such as group chats or verbal complaints, which makes it unstructured and difficult for wardens or mess staff to analyze and act upon.

To improve food quality, transparency, and accountability, the Hostel Committee requires a **simple digital system** to collect feedback and present insights in a structured manner.

---

## Objective

To build a **Smart Mess Feedback & Menu Portal** that enables students to share feedback easily and helps wardens analyze mess performance efficiently.

---

## Solution Overview

**Messy Matters** is a web-based application designed to provide a centralized platform where:

- Students can view mess menus and submit feedback.
- Wardens can monitor ratings, complaints, and meal-wise performance.
- Menu updates can be managed digitally in real time.

---

## Features

### Student Side
- Display of **daily and weekly mess menu**
- Meal-wise rating system:
  - Breakfast
  - Lunch
  - Snacks
  - Dinner
- Feedback submission with:
  - Anonymous option
  - Named submission option
- Simple and user-friendly interface

---

### Warden / Mess Staff Dashboard
- Secure warden login
- Dashboard to view:
  - Average meal ratings
  - Frequent complaints
  - Meal-wise comparison
- Menu update functionality
- Clear separation between public view and admin view

---

## Benefits

- Provides structured and organized feedback
- Encourages honest feedback through anonymity
- Helps wardens make data-driven decisions
- Improves food quality and student satisfaction
- Eliminates dependency on informal complaint channels

---

## Technology Stack

- Frontend: Next.js, React
- Backend: Next.js API Routes
- Database: MongoDB Atlas (with local JSON fallback)
- Data Visualization: Chart.js
- Deployment: Vercel

---

## Conclusion

**Messy Matters** offers a practical and scalable solution for hostel mess management by transforming unstructured complaints into actionable insights, thereby improving transparency, efficiency, and overall dining experience.
